    archinstall é uma biblioteca auxiliar para instalar o Arch Linux. Ele é empacotado com diferentes instaladores pré-configurados, como um instalador "guiado".

    Este documento não discute o uso do archinstall como uma biblioteca Python; para isso, veja a documentação oficial.

    Atenção:
    • archinstall armazena todas as senhas de criptografia de usuário e de disco (secundário) em texto simples. [1] [2]
    • archinstall oferece padrões diferentes do processo normal de instalação. Ao usar um sistema instalado com archinstall, mencione isso nas solicitações de suporte e forneça /var/log/archinstall/install.log.
    Nota: archinstall tem suporte experimental para BIOS via GRUB.

    Executando o instalador

    Primeiro, adquira e inicialize a mídia live conforme descrito em Guia de instalação#Pré-instalação. O pacote archinstall faz parte da mídia live e pode ser executado diretamente: 

    # archinstall

    O instalador guiado irá realizar (consultar) as seguintes etapas:

    1. configurar o locale;
    2. selecionar os mirrors;
    3. particionar os discos;
    4. formatar as partições;
    5. habilitar criptografia de disco (opcional);
    6. definir o hostname;
    7. definir a senha do usuário root;
    8. instalar um gerenciador de boot.
    Atenção: Deixar a senha do usuário root vazia desabilita a conta root, usando sudo para escalonamento de privilégios. Como isso pode fazer com que se você se bloqueie do sistema, isso geralmente não é recomendado. Veja Sudo#Desabilitar o login do root.
    Nota: O instalador pode configurar redes a cabo no sistema instalado usando Systemd-networkd ou pode copiar a configuração usada na ISO. Isso quer dizer que, se você configurar uma rede sem fio usando Iwd#iwctl, sua configuração, incluindo a senha da rede, serão copiadas para o sistema instalado. Também copia a configuração para redes cabeadas que estão presentes na ISO.

    Pacotes adicionais podem ser instalados ao especificá-los depois da mensagem Write additional packages to install ("Escreva pacotes adicionais para instalar").

    Uma vez que a instalação foi concluída, um texto verde deve aparecer dizendo que é seguro reiniciar, que é também o comando que você usa para reiniciar.

    Perfis

    archinstall inclui perfis (profiles), ou conjuntos de pacotes, e opções pré-configuradas que podem ser instaladas junto ao sistema base.

    Atenção: Perfis enviados são específicos para archinstall e não são suportados por mantenedores de pacotes. Os usuários são aconselhados a verificar os detalhes de cada perfil antes de usá-lo.
