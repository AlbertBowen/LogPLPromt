    Status de tradução: Esse artigo é uma tradução de Reset lost root password. Data da última tradução: 2022-10-25. Você pode ajudar a sincronizar a tradução, se houver alterações na versão em inglês.

    Este guia irá mostrar-lhe como redefinir uma senha de root esquecida. Vários métodos estão listados para ajudá-lo a realizar isso.

    Atenção: Um invasor pode usar alguns dos métodos mencionados abaixo para entrar em seu sistema. Não importa o quão seguro o sistema operacional seja ou como as senhas são boas, ter acesso físico equivale a carregar um sistema operacional alternativo e expor seus dados, a menos que você use criptografia de dados em repouso.

    Usando o sudo

    Se você instalou o sudo e configurou permissões para o grupo wheel ou um usuário cuja senha você se lembra, você pode alterar a senha do root executando sudo passwd root.

    Usando o shell de depuração

    1. Anexe systemd.debug_shell aos parâmetros do kernel.
    2. Isso vai fazer uma inicialização normal, mas inicia debug-shell.service, o qual executa um shell de root (/bin/sh) no tty9. Pressione Ctrl+Alt+F9 para acessá-lo.
    3. Use o comando passwd para criar uma nova senha para o usuário root.
    4. Quando finalizar, pare debug-shell.service.

    Usando bash como init

    1. Anexe o parâmetro do kernel init=/bin/bash à entrada de inicialização do seu gerenciador de boot.
    2. Seu sistema de arquivos raiz está montado como somente leitura agora, então remonte-o como leitura/escrita: mount -n -o remount,rw /.
    3. Use o comando passwd para criar uma nova senha para o usuário root.
    4. Reinicialize digitando reboot -f e não perca sua senha novamente!
    Nota: Alguns teclados podem não ser carregados corretamente pelo sistema init com este método e você não poderá digitar nada no prompt do bash. Se for esse o caso, você terá que usar outro método.

    Usando um LiveCD

    Com um LiveCD, alguns métodos estão disponíveis: alterar a raiz e usar o comando passwd, ou apagar a entrada do campo da senha diretamente editando o arquivo de senha. Qualquer LiveCD compatível com Linux pode ser usado, embora para mudar de raiz deve corresponder ao tipo de arquitetura instalada. Aqui, descrevemos apenas como redefinir sua senha com chroot, uma vez que a edição manual do arquivo de senha é significativamente mais arriscada.

    Alterando a raiz

    1. Inicie o LiveCD e monte a partição raiz de seu sistema principal.
    2. Use o comando passwd --root PONTO_DE_MONTAGEM NOME_USUÁRIO para definir a nova senha (você não será perguntado sobre a anterior).
    3. Desmonte a partição raiz.
    4. Reinicie e insira sua nova senha.
