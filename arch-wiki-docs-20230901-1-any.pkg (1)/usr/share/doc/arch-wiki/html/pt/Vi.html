    Do Wikipedia:

    O vi é um editor de texto orientado a tela criado originalmente para o sistema operacional Unix. O subconjunto portátil do comportamento do vi e dos programas com base nele, e a linguagem do ex editor suportada nesses programas, é descrito (e, portanto, padronizado) pela Single Unix Specification e POSIX.

    Instalação

    Instale o pacote vi.

    Software estilo Vi

    As combinações de teclas e modos do vi foram recriados em vários outros softwares:

    • editores de texto, o mais popular dos quais sendo o Vim
    • gerenciadores de arquivo como o ranger e Vifm
    • O Readline biblioteca de entrada (usado no Bash)
    • shells como Zsh
    • navegadores web como Luakit, qutebrowser or vimb; para Firefox e Chromium tem as extensões de navegador disponíveis.
    • a maioria das tiling window manager pode ser configurado para controle estilo vi
