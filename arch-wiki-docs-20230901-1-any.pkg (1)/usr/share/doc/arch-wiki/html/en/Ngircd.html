    From ngIRCd:

    ngIRCd is a free, portable and lightweight Internet Relay Chat server for small or private networks, developed under the GNU General Public License.

    Installation

    Install the ngircdAUR package.

    Importing the GPG key

    ngircd follows the official package format, the gpg keys to verify the source are located under keys/pgp directory, these must be imported before you are able to build the package, see GnuPG#Import a public key.
