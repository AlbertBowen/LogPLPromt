    This laptop comes in an O or D model:

    O = Onboard Vega graphics

    D = Onboard + Dedicated nVidia MX450 2GB GPU

    Installation

    In order to install, Secure Boot must be disabled in the UEFI. You can enter it by pressing F2 (or Fn+F2) during boot. In order to access the boot-menu allowing you to choose a non standard boot device (like USB devices), press F12 (or Fn+F12) during boot.

    Firmware

    While Lenovo is known to support fwupd in general, the Yoga Slim 7 has not been added yet.

    UEFI

    The UEFI can be accessed at boot by pressing F2, minimal options are available.

    Updates are currently provided by Lenovo as Windows binaries.

    Display

    The laptop comes with an optional 2800x1800 90hz glossy display: recommended settings for Gnome are 200% scale and 85% font scale, KDE scaling is customizable enough to handle the resolution.

    For more details, see HiDPI.

    Graphics

    The integrated Vega graphics are decent, on the D model the dedicated NVIDIA graphics can be disabled in the BIOS under the "UMA Graphics" setting.

    When using the dedicated NVIDIA graphics it is recommended to keep the chip off when not in use using one of the NVIDIA Optimus solutions

    Power management

    Suspend

    s0ix is the only supported sleep mode, this mode works well in kernel 5.16 onwards although a slight drain is observed while the machine is in a sleep state.

    Hibernation

    Untested although hibernation should work using kernels 5.16 onwards.

    Battery Saving

    Using GNOME, the battery saving profile can be changed which is reflected in the UEFI

    • Battery Life − Battery Saver
    • Balanced − Intelligent Cooling
    • High Performance − Performance

    It is recommended to install TLP for battery saving.

    This laptop supports the amd_pstate frequency driver, see CPU frequency scaling once amd_pstate is activated the cores can run as low as 400mhz reducing the battery consumption when idle.

    A custom resolution of 2880x1800 at 60hz can be added to reduce drain from running the display at 90hz.

    Note:
    • When operating in Balanced power mode on AC power, the machines fan will often be spinning.
    • Battery conservation mode and Rapid Charge mode can be turned on from Windows using Lenovo Vantage program. It will stay turned on after booting Linux.
