    • Mirrors

    These mirrors are not listed by default in /etc/pacman.d/mirrorlist, therefore not distributed within pacman-mirrorlist.

    Note: The mirror list only contains the mirror URL, these should be modified by appending $repo/os/$arch to the end of the URL when adding them to /etc/pacman.d/mirrorlist

    Up to date

    Up to date mirrors have the latest packages, and remain synchronised with the official arch repositories. These mirrors can be used like any official mirror distributed by pacman-mirrorlist.

    Worldwide

    • https://cloudflaremirrors.com/archlinux/ - CloudFlare runs a quasi-mirror service that fronts existing mirrors on their global CDN (might not work correctly with pacman, see FS#67865)

    Out of date

    Out of date mirrors do not have the latest packages, these are either no longer maintained, or designed to distribute older packages which are no longer available within the official mirrors.

    Worldwide

    • https://sourceforge.net/projects/archlinux/files/ - ISO files only; Does not have any releases since 2006. Use it only for getting older ISOs.
