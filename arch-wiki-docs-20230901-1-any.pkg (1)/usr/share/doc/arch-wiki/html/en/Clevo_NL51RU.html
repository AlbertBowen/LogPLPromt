    The NL51RU is a device by the taiwanese OEM manufacturer Clevo. It is sold as System76 Pangolin, Tuxedo Aura 15. The hardware is configurable and includes an AMD Ryzen 5 4500U or AMD Ryzen 7 4700U, AMD radeon 6 core integrated graphics, up to 64GB RAM @3200MHz, SSD (SATA or NVMe), microSD card reader, 3g/LTE modem and a HDMI output.

    Note: Even though case has slots for microSD card and SIM card, it is possible that modem and reader are not installed.

    Firmware

    fwupd is supported. Testing showed System firmware and SSD firmware updateable.

    Webcam

    The webcam needs to be activated by pressing Fn+F10, otherwise device will not be available.

    Sound

    Sound mostly works out of the box, tested with ALSA and PulseAudio. Wrong sound card may be selected by default. Microphone seems to be working out of the box as well.

    Touchpad

    The touchpad works out of the box. Scrolling and multi finger touches work, as well as left/right clicks. Gestures seem to be working, tested with libinput-gesturesAUR.

    WiFi

    WiFi seems to work well. When tuxedo-keyboard-dkmsAUR is installed, Fn+F11 toggles airplane mode.

    Function keys

    Key Visible?1 Marked?2 Effect
    Fn+F1 Yes Yes XF86TouchpadToggle
    Fn+F2 No Yes Disables display
    Fn+F3 Yes Yes XF86AudioMute
    Fn+F5 Yes Yes XF86AudioLowerVolume
    Fn+F6 Yes Yes XF86AudioRaiseVolume
    Fn+F7 Yes Yes Seems to have same effect as Super+p
    Fn+F8 Yes Yes XF86MonBrightnessDown
    Fn+F9 Yes Yes XF86MonBrightnessUp
    Fn+F10 No Yes Toggles webcam
    Fn+F11 No Yes Toggles airplane mode
    Fn+F12 No Yes Suspend device
    Fn+` Yes Yes XF86AudioStop
    Fn+Numpad/ No Yes Switch keyboard backlight color
    Fn+Numpad* No Yes Toggle keyboard backlight
    Fn+Numpad- No Yes Decrease keyboard backlight brightness
    Fn+Numpad+ No Yes Increase keyboard backlight brightness
    Fn+1 No No Seems to toggle fan at max speed
    1. The key is visible to xev and similar tools
    2. The physical key has a symbol on it, which describes its function

    To make use of some of the Fn keys (Airplane mode, keyboard backlight), kernel module needs to be added. Installing tuxedo-keyboard-dkmsAUR from the AUR was tested and enables these keys after reboot.
