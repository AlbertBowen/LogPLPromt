    Help
    From ArchWiki

    This category groups categories used for maintenance purposes. All of them are hidden through Template:Hidden category. More tracking categories whose pages were not created yet might be found in Special:TrackingCategories.

    These categories should not be used directly, pages are only added to them either automatically or via templates transclusion.

    See also ArchWiki:Statistics#Maintenance statistics which shows how many pages are in each subcategory.
