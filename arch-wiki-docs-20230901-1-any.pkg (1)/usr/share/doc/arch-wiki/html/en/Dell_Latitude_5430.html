    Configuration

    SmartCard Reader

    Works out-of-the-box with ccid & opensc . Read Smartcards for more info.

    NFC Reader

    Although it is detected by pc_scan, RF field is not present thus no signal is detected. For enabling the NFC, this python script is needed:

    https://github.com/jacekkow/controlvault2-nfc-enable

    python and python-pyusb are needed.

    Finger print

    libfprint-2-tod1-broadcomAUR is needed to enable the fprint.
