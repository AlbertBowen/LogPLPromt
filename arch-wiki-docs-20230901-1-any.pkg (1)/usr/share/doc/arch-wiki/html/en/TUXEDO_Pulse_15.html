    Function keys

    Most of the function keys should be recognised automatically once tuxedo-keyboard-dkmsAUR is installed.

    Key Visible? Marked? Effect
    Fn+Esc No Yes Toggles Fn lock
    Fn+F1 No Yes XF86AudioMute
    Fn+F2 No Yes XF86LowerVolume
    Fn+F3 No Yes XF86RaiseVolume
    Fn+F4 No No
    Fn+F5 No No
    Fn+F6 No Yes Marked to launch tuxedo-control-center but is recognized by xev as Super_L
    Fn+F7 No Yes XF86MonBrightnessDown
    Fn+F8 No Yes XF86MonBrightnessUp
    Fn+F9 No Yes Toggles the touchpad on/off
    Fn+F10 No Yes XF86RFKill
    Fn+F11 No Yes Recognized as Super_L
    Fn+F12 No Yes XF86Sleep

    Video

    The Pulse 15 comes with a Ryzen 7 4800H CPU, which includes a Renoir GPU. The laptop works with the open source xf86-video-amdgpu driver.

    Keyboard backlight

    TUXEDO furnishes a driver for this. It can be installed from source or with tuxedo-keyboard-dkmsAUR.

    Power management

    It is advisable to install the official tuxedo-control-center-binAUR to let it control the fan speed and CPU frequency. The package will install and enable tccd.service, which is all that is needed for the laptop to activate the power management options chosen.

    To actually change them, select different profiles according to the battery status (running on battery power or plugged in), a GUI binary is also furnished, tuxedo-control-center.
