    Accessibility

    The appearance of the BIOS is pretty simple (grey and blue), so it might work well with OCR software. It does not have mouse support

    Note: Blind users should request the help of a sighted person to change BIOS settings

    Firmware

    fwupd does not support this device yet.

    GPU

    The NVIDIA GPU / AMD iGPU combo supports PRIME

    Function keys

    Key Visible?1 Marked?2 Effect
    Fn+F8 Yes Yes XF86AudioMute
    Fn+DownArrow Yes Yes XF86AudioLowerVolume
    Fn+UpArrow Yes Yes XF86AudioRaiseVolume
    Fn+PageDown Yes Yes XF86AudioPrev
    Fn+PageUp Yes Yes XF86AudioPlay
    Fn+End Yes Yes XF86AudioNext
    Fn+F9 Yes Yes Step the keyboard backlight level down
    Fn+F10 Yes Yes Step the keyboard backlight level up
    Fn+LeftArrow Yes Yes XF86MonBrightnessDown
    Fn+RightArrow Yes Yes XF86MonBrightnessUp
    Fn+F3 Yes Yes XF86RFKill, will hard-block wifi and soft-block Bluetooth. Press again to disable
    Fn+F6 Yes Yes Turns the screen off, any keyboard action will wake it up
    Fn+F4 Yes Yes XF86Sleep
    Fn+F11 Yes Yes Pause
    Fn+F12 Yes Yes ScrollLock
    Fn+Del Yes Yes Insert
    1. The key is visible to xev and similar tools
    2. The physical key has a symbol on it, which describes its function
    3. systemd-logind handles this by default
