    From ArchWiki

    surf — це простенький веб переглядач з ядром WebKit/GTK. В ньому можна дивитись веб сайти та переходити за посиланнями. Він підтримує протокол XEmbed а значить, його можна вставляти в інші програми. Ці програми можуть вказувати, яку адресу показувати, задавши XProperties браузера.

    Встановлення

    Встановіть пакунок surfAUR. Або пакунок surf-gitAUR — найновішу версію, що перебуває в розробці.

    Також можете встановити dmenu щоб мати поле URL адреси.

    Налагодження

    Surf Налагоджується через файл config.h. В код програми включений приклад — файл config.def.h. По ньому ви зрозумієте, як налаштувати програму.

    Як із іншими програмами, на зразок dwm, вам варто використати „Систему побудови Arch“ (Arch Build System) (ABS) та підготувати власний PKGBUILD з вашими файлами налаштувань.

    Покращення

    Patches & additional features

    There are many user-created patches available from the offical site that greatly extend the functionality of surf. Patches can be applied to both the source surf.c file and the config.h file: 

    $ cd src/surf-version/
$ patch -p1 < path/to/patch.diff

    Вкладки

    Щоб мати вкладки, поєднайте Surf із програмою tabbed-gitAUR.

    Використання: 

    $ tabbed surf -e

    Щоб вкладки працювали схожим чином до Firefox або Chromium: щоб програма закривалась коли закриється остання вкладка, використайте: 

    $ tabbed -c surf -e

    Прочитайте сторінку man посібника tabbed(1) щоб дізнатися більше.

    Виправлення помилок

    Розмиті шрифти в Github

    Встановіть gnu-free-fonts або додайте наведений нижче текст до ~/.config/fontconfig/fonts.conf всередині тегу fontconfig-tags: 

     <selectfont>
   <rejectfont>
     <pattern>
       <patelt name="family">
         <string>Clean</string>
       </patelt>
     </pattern>
   </rejectfont>
 </selectfont>
