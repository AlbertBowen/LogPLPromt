    Nitrogen est une application graphique (en GTK2) fournissant un aperçu des images d'un répertoire et permettant de les définir comme fond d'écran.

    Installation

    Vous pouvez installer le paquet nitrogen avec pacman.

    Utilisation

    Utilisez nitrogen --help pour plus de détails. Les exemples suivants vous permettront de commencer :

    Sélectionner le fond d'écran

    Pour voir et sélectionner (récursivement par défaut) le fond d'écran désiré depuis un répertoire spécifique, utilisez: 

    $ nitrogen /path/to/image/directory/

    Non-récursivement: 

    $ nitrogen --no-recurse /path/to/image/directory/

    Restaurer le fond d'écran

    Pour retrouver votre fond d'écran au démarrage de votre session, ajoutez la ligne suivante dans votre script de lancement de session: 

    $ nitrogen --restore &

    Dépannage

    Freeze avec double-écran

    Supprimez la configuration actuelle (voir [1]): 

    $ rm -r ~/.config/nitrogen/
