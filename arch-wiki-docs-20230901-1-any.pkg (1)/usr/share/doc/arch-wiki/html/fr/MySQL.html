    État de la traduction: Cet article est la version francophone de MySQL. Date de la dernière traduction: 2022-01-15. Vous pouvez aider à synchroniser la traduction s'il y a eu des changements dans la version anglaise.

    MySQL est une base de données SQL largement répandue, multi-threads et multi-utilisateurs, développée par Oracle.

    Arch Linux favorise MariaDB, un dérivé de MySQL développé par la communauté, en vue d'une compatibilité immédiate. MySQL d'Oracle a été abandonné dans l'AUR : mysqlAUR. Un autre fork visant à être totalement compatible est Percona Server, disponible sous percona-server.

    Le moteur de stockage InnoDB d'Oracle a également été bifurqué par Percona sous le nom de XtraDB. Ce «fork» est utilisée à la fois par MariaDB et Percona Server.

    Outils graphiques

    • phpMyAdmin — Interface web de MySQL, écrite en PHP.
    https://www.phpmyadmin.net || phpmyadmin
    • MySQL Workbench — Outil visuel unifié pour les architectes de bases de données, les développeurs et les DBA. Développé par Oracle et dont le fonctionnement n'est pas garanti avec MariaDB.
    https://www.mysql.com/products/workbench/ || {Pkg}}

    Pour les outils prenant en charge plusieurs SGBD, consultez List of applications/Documents#Database tools.

    https://www.mycli.net || mycliAUR
