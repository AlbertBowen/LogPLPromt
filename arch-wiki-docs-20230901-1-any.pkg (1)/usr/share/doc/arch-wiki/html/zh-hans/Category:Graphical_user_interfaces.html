    本分类包含 图形用户界面 (GUI) 设置的相关文章，GUI 程序属于 Category:Applications.

    Desktop environment (简体中文)
    通过整合各种组件，提供一个完整的图形系统。
    Window manager (简体中文)
    控制图形窗口位置和外观的系统程序。
    Display manager (简体中文)
    图形登录管理器，在显示服务器上启动会话。
