    X 視窗系統是圖形介面的基石，Xorg 專案就是它的開源碼實作版本。桌面環境 (如 Enlightenment, GNOME, KDE, LXDE 和 Xfce) 提供了完整的圖形環境；另外還有各式各樣的視窗管理員可選，它們可以「獨立」運作以節省系統資源。顯示管理員則提供圖形登入介面。
