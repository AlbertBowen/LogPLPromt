    From ArchWiki

    Arch Linux поставляется с двумя опциями для управления межсетевым экраном, которые отключены по умолчанию. Стандартное ядро Linux включает в себя фреймворк фильтрации пакетов netfilter, которым можно управлять одной из следующих утилит:

    • iptables – традиционная утилита для управления межсетевым экраном из пользовательского пространства. Управлять конфигурацией можно непосредственно с помощью утилит в пользовательском пространстве, также можно установить утилиту с графическим интерфейсом.
    • nftables – более новый проект, заменяющий фреймворк iptables. Совмещает простой синтаксис, широкие возможности и большую производительность в сравнении с iptables.
