    Тип PCI/USB ID Работает?
    Дисплей (Intel) 8086:3184 Да
    Wi-Fi 8086:31dc Да
    Bluetooth 8087:0aaa Да
    Звук 8086:3198 Да
    Акселерометр (BMI160) Нет
    Камера (Основная 5M) 058f:5608 Да
    Камера (Фронтальная 2M) 0bda:5609 Да
    Клавиатура & Тачпад 17ef:60fb Да
    Тачскрин 04f3:2bd6 Да

    Характеристики

    Ноутбук-планшет 2 в 1.

    Pentium N5030

    • Дисплей: UHD Graphics 605 - 10.3" 1920x1200
    • Процессор: Intel Pentium N5030 1.1 Ghz
    • Оперативная память: 8GB
    • Внутренняя память: 128GB eMMC

    Celeron N4020

    • Дисплей: UHD Graphics 600 - 10.3" 1920x1200
    • Процессор: Intel Celeron N4020 1.1 Ghz
    • Оперативная память: 4GB
    • Внутренняя память: 128GB/64GB eMMC

    Звук

    Работает из коробки.

    Дисплей

    Т.к. дисплей в данном аппарате стоит нестандартный, он имеет свойство поворачиваться на 90 градусов, считая данный поворот основным. Чтобы его повернуть, необходимо в параметры ядра написать значения: 

    fbcon=rotate:1 video=DSI-1:panel_orientation=right_side_up

    Первый параметр (fbcon) отвечает за поворот консоли, второй параметр (video) отвечает за поворот с графических интерфейсах, например Plymouth. Также для полной работоспособности дисплея, необходимо установить пакет xf86-video-intel, и добавить драйвер в mkinitcpio. 

    /etc/mkinitcpio.conf
    MODULES=(... i915 ...)

    Тачскрин

    Поворот тачскрина с помощью конфига X. 

    /etc/X11/xorg.conf.d/40-touchscreen.conf
    Section "InputClass"
        Identifier      "Lenovo Touchscreen"
        MatchProduct    "ELAN901C:00 04F3:2BD6"
        Driver "libinput"
        Option  "TransformationMatrix"  "0 1 0 -1 0 1 0 0 1"
EndSection

    Драйверы

    Т.к. некоторые прошивки не поставляются пакетом linux-firmware, в процессе сборки initramfs могут быть ошибки: 

    WARNING: Possibly missing firmware for module: XXX

    Подробнее смотрите Mkinitcpio (Русский)#Possibly missing firmware for module XXXX

    Акселерометр

    В данном устройстве стоит некий BMI160 . На данный момент неизвестно, как к нему подключиться, т.к. iio не видит его вовсе. Из-за этого автоповорот дисплея не работает.
