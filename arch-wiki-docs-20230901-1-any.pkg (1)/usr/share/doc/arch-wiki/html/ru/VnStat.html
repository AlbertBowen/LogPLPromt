    vnStat это легковесная программа для мониторинга сетевого трафика. Лог ведется по выбранному сетевому интерфейсу, а статистику можно посмотреть используя командную строку.

    Установка

    Установите пакет vnstat.

    Настройка

    Запустите и включите службу vnstat.service.

    Выберите нужный сетевой интерфейс и измените переменную Interface соответственно в файле /etc/vnstat.conf. Чтобы посмотреть список всех интерфейсов, которые видит vnstat, используйте vnstat --iflist.

    Чтобы начать мониторинг определенного интерфейса, который не упоминался в файле конфигурации при запуске демона, нужно сначала инициализировать базу данных. У каждого интерфейса своя отдельная база. Команда для её инициализации для интерфейса eth0 выглядит так: 

    # vnstat --add -i eth0

    Перезапустите vnstat.service после добавления нового интерфейса.

    Использование

    Показать статистику: 

    # vnstat --query

    Показать статистику в реальном времени: 

    # vnstat --live

    Посмотреть доступные опции: 

    # vnstat --help

    Или, для полного текста справки: 

    # vnstat --longhelp

    Программа vnstati(1), являющаяся частью пакета vnstat, предоставляет красивый интерфейс просмотра данных.
