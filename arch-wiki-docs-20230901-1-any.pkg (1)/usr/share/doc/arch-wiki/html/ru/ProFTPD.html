    ProFTPD (Pro FTP daemon) — это многофункциональный и гибко настраиваемый FTP-сервер.

    Установка

    Установите пакет proftpdAUR, затем запустите и включите службу proftpd.service.

    Настройка

    Расположение файла конфигурации — /etc/proftpd.conf. На сайте проекта доступна подробная документация.

    Анонимный доступ

    Чтобы анонимный доступ работал при использовании /bin/false в качестве командной оболочки для пользователя ftp (конфигурация по умолчанию), добавьте строку RequireValidShell off в файле /etc/proftpd.conf. Без этого попытки анонимного входа будут приводить к ошибке 530.
