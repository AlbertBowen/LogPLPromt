    Из домашней страницы проекта:

    Webmin - это веб-интерфейс для системного администрирования Unix. Используя современный веб-браузер, вы можете настроить учётные записи пользователей, Apache, DNS, файл-сервер и многое другое. Webmin избавляет от необходимости вручную редактировать конфигурационные файлы Unix, такие как /etc/passwd, а также позволяет управлять системой из консоли или удалённо. Смотрите страницу стандартных модулей Webmin или скриншоты.

    Установка

    Установите пакет webminAUR.

    Настройка

    Чтобы иметь доступ к Webmin с удаленного компьютера, настройте ваш межсетевой экран на пропуск входящего трафика на порт 10000. Вы также можете захотеть разрешить доступ только с определённых IP-адресов.

    Изменение порта

    Для изменения порта задайте нужное значение переменной port в файле /etc/webmin/miniserv.conf. 

    /etc/webmin/miniserv.conf
    [...]
port=10000
[...]

    Слушать только localhost

    Чтобы Webmin принимал трафик только с localhost, добавьте в /etc/webmin/miniserv.conf: 

    /etc/webmin/miniserv.conf
    bind=127.0.0.1

    Использование

    Запустите службу webmin.service, которая управляет демоном Webmin. По желанию включите её автозапуск при загрузке системы.

    В веб-браузере наберите адрес сервера, на котором установлен Webmin, указав порт 10000: 

    https://хост:10000

    Для доступа нужно будет ввести root-пароль от сервера.
