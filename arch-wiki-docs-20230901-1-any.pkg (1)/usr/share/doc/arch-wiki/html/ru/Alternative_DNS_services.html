    Cisco Umbrella (ранее OpenDNS)

    OpenDNS предоставлял бесплатные альтернативные серверы DNS. Выкуплен Cisco в ноябре 2016 и продолжает предоставлять OpenDNS под новым брендом "Umbrella", который сфокусирован на обеспечении и анализе безопасности, веб-фильтрации. Старые DNS сервера работают до сих пор, но теперь они преднастроенны чтобы блокировать контент для взрослых: 

    208.67.222.222
208.67.220.220
2620:0:ccc::2
2620:0:ccd::2

    Cloudflare

    Cloudflare предоставляет сервис, обязующийся никогда не записывать запрашиваемые IP-адреса на диск и стирать все логи в течение 24 часов, за исключением предоставления данных APNIC labs в исследовательских целях. APNIC и Cloudflare обязуются обрабатывать все данные в соответствии с высокими стандартами конфиденциальности в своих договором об исследовании. 

    1.1.1.1
1.0.0.1
2606:4700:4700::1111
2606:4700:4700::1001

    Comodo

    Comodo предоставляет другой набор IPv4 DNS серверов, с опциональной веб-фильтрацией. Подразумевается, что эта функция заключается в том, что служба перехватывает запросы. 

    8.26.56.26 
8.20.247.20

    DNS.WATCH

    DNS.WATCH фокусируется на нейтралитете и безопасности. Предоставляет два сервера поддерживающие DNSSEC и раположеные в Германии. Обратите внимание, что они приветствуют коммерческое спонсорство. 

    84.200.69.80                # resolver1.dns.watch 
84.200.70.40                # resolver2.dns.watch
2001:1608:10:25::1c04:b12f  # resolver1.dns.watch
2001:1608:10:25::9249:d69b  # resolver2.dns.watch

    Google

    DNS-серверы Google могут быть использованы в качестве альтернативы: 

    8.8.8.8
8.8.4.4
2001:4860:4860::8888
2001:4860:4860::8844

    Mullvad

    Mullvad предоставляет публичный DNS сервер, неведущий логи. 

     193.138.218.74

    OpenNIC

    OpenNIC предоставляет свободные от цензуры DNS-серверы, расположенные по всему миру. Полный список публичных серверов доступен на servers.opennic.org Список ближайших к вам серверов генерируемый для оптимальной производительности расположен на главной странице.

    Чтобы извлечь список ближайших DNS-серверов существует API возвращающий список DNS-серверов на основе предоставленных URL параметров. Например, для получения списка 200 ближайших IPv4 серверов, можно воспользоваться ссылкой вида https://api.opennicproject.org/geoip/?list&ipv=4&res=200&adm=0&bl&wl.

    В качестве альтернативы, вы можете использовать приведенные ниже Anycast серверы; несмотря на надежность, их задержка cильно колеблется.

    Worldwide Anycast: 

    185.121.177.177
169.239.202.202
2a05:dfc7:5::53
2a05:dfc7:5::5353
    Примечание: The use of OpenNIC DNS servers will allow host name resolution in the traditional Top-Level Domain (TLD) registries, but also in OpenNIC or afiliated operated namespaces: .o, .libre, .dyn...
    Совет: The tool opennic-up — automates the renewal of the DNS servers with the most responsive OpenNIC servers
    https://github.com/kewlfft/opennic-up || opennic-upAUR

    Quad9

    Quad9 свободный DNS сервис основанный IBM, Packet Clearing House и Global Cyber Alliance; уникальная фишка сервиса - чёрный список, позволяющий избежать разрешения известных вредоносных доменов. Anycast серверы, расположенные по всему миру, перечислены ниже.

    "Безопасные", с чёрным списком и поддержкой DNSSEC: 

    9.9.9.9
149.112.112.112
2620:fe::fe
2620:fe::9

    Без чёрного списка, без DNSSEC: 

    9.9.9.10
149.112.112.10
2620:fe::10

    UncensoredDNS

    UncensoredDNS бесплатный DNS сервис без цензуры. Он управляется частным лицом и состоит из одной anycast сети, обслуживаемой несколькими серверами, и одного узла одноадресной рассылки, размещенного в Дании. 

    91.239.100.100   # anycast.censurfridns.dk
89.233.43.71     # unicast.censurfridns.dk
2001:67c:28a4::  # anycast.censurfridns.dk
2a01:3a0:53:53:: # unicast.censurfridns.dk
    Примечание: Its servers listen to port 5353 as well as the standard port 53. This can be used in case your ISP hijacks port 53.

    Yandex

    Yandex.DNS есть серверы в России, в Восточной и Западной Европе и три режима, Базовый, Безопасный и Семейный.

    Базовый - без фильтрации трафика: 

    77.88.8.8
77.88.8.1
2a02:6b8::feed:0ff
2a02:6b8:0:1::feed:0ff

    Безопасный - с заащитаой от заражённых и мошеннических сайтов: 

    77.88.8.88
77.88.8.2
2a02:6b8::feed:bad
2a02:6b8:0:1::feed:bad

    Family - всё вышеперечисленное и блокировка сайтов для взрослых: 

    77.88.8.7
77.88.8.3
2a02:6b8::feed:a11
2a02:6b8:0:1::feed:a11
