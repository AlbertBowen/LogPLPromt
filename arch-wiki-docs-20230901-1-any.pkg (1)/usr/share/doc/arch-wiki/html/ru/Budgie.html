    Budgie — среда рабочего стола, в прошлом — проект, разрабатываемый в рамках Solus, ставший независимым под управлением организации Buddies of Budgie в январе 2022 года. Она использует GTK 3 для виджетов и написана на C и Vala. На момент релиза версии Budgie 10 доступен только сеанс Xorg.

    Установка

    Установите группу пакетов budgie, чтобы установить все основные компоненты рабочего стола. Пакеты также могут быть установлены по отдельности — используйте пакет budgie-desktop для последней стабильной версии или budgie-desktop-gitAUR для текущего git main. Также параллельно могут быть установлены budgie-screensaver для поддержки блокировки рабочего стола и budgie-control-center для модификации системных настроек. Следующие пакеты опциональны, однако добавляют дополнительные функции в рабочий стол:

    • budgie-desktop-view: Официальная реализация иконок рабочего стола
    • budgie-backgrounds: Официальный набор стандартных фонов рабочего стола
    • network-manager-applet: Управление сетью из панели
    • materia-gtk-theme или arc-gtk-theme: Рекомендованные темы GTK
    • papirus-icon-theme: Тема иконок по умолчанию

    Дополнительные апплеты, разработанные командой Ubuntu Budgie доступны в пакете budgie-extras - однако имейте в виду, что этот пакет также модифицирует уже имеющиеся функции и может создать проблемы.

    Запуск

    Выберите сеанс Budgie Desktop в экранном менеджере или измените xinitrc, добавив Budgie Desktop: 

    ~/.xinitrc
    export XDG_CURRENT_DESKTOP=Budgie:GNOME
exec budgie-desktop

    Использование

    Вы можете видеть уведомления, устанавливать системную громкость и громкость для приложений, просматривать календарь, и просмотреть проигрываемые видео и музыку при помощи боковой панели под названием "Raven". Доступ к разделу "Уведомления" осуществляется нажатием Super+n или щелчком по апплету "Уведомления". Доступ к разделу "Апплеты" может быть получен нажатием Super+a. Raven также можно открыть нажатием на апплет "Raven Trigger", после чего он откроется на последней выбранной панели.

    Темы

    Budgie использует GTK 3 для своих элементов интерфейса, и таким образом, поддерживается многими темами GTK. Budgie также предоставляет встроенную тему, которая применяется только к собственным элементам, таким как панели и Raven, и может быть настроена в настройках рабочего стола Budgie. Темы иконок и курсоров настраиваются там же.

    Настройка

    Настройка рабочего стола Budgie выполняется при помощи встроенного приложения "Настройки рабочего стола Budgie", а изменения в системные настройки выполняются через budgie-control-center.

    Изменение раскладки кнопок

    Изменить раскладку кнопок на окнах можно при помощи dconf, dconf-editor или gsettings.

    Например: 

    $ gsettings set com.solus-project.budgie-wm button-layout 'close,minimize,maximize:appmenu'
$ gsettings set com.solus-project.budgie-helper.workarounds fix-button-layout 'close,minimize,maximize:menu'

    Использование иного менеджера окон

    Budgie не поддерживает использование другого оконного менеджера.
