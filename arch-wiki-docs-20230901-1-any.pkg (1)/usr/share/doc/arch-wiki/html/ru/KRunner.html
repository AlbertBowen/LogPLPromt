    • KDE (Русский)

    KRunner — приложение, встроенное в Plasma 5, для быстрого запуска функций и команд и использующее систему плагинов для настройки доступных для использования функций.

    Установка

    Установите пакет krunner.

    Использование

    Открыть KRunner в Plasma можно либо через контекстное меню рабочего стола, либо сочетанием клавиш Alt+Space или Alt+F2. В некоторых рабочих пространствах, например, на пустом рабочем столе, начало ввода текста автоматически вызовет KRunner.

    Открытие KRunner клавишей Meta

    Чтобы сделать Meta горячей клавишей для запуска KRunner, выполните 

    $ kwriteconfig5 --file kwinrc --group ModifierOnlyShortcuts --key Meta "org.kde.krunner,/App,,toggleDisplay"

    Для применения изменения в текущем сеансе отправьте D-Bus сигнал процессу KWin: 

    $ qdbus org.kde.KWin /KWin reconfigure

    Если вы не используете D-Bus, перезапустите KWin: kwin_x11 --replace в X11 или kwin_wayland --replace в Wayland.

    Переключение между окнами

    Если вы хотите использовать KRunner только для переключения между открытыми окнами, можно отключить в настройках все лишние модули, кроме модуля «Окна». Однако имейте в виду, что модули поиска также используются в виджете запуска приложений и в эффекте обзора. Если полное отключение вам не подходит, можно использовать другой способ.

    1. Откройте Параметры системы > Рабочая среда > Комбинации клавиш > Специальные действия.
    2. Выберите пункт Изменить > Создать > Глобальная комбинация клавиш > Команда D-Bus.
    3. Поставьте галочку справа от имени созданного вами действия.
    4. На вкладке Активация выберите желаемое сочетание клавиш.
    5. На вкладке Действие пропишите следующие параметры: 
       - Приложение : org.kde.krunner
 - Объект     : /App
 - Функция    : querySingleRunner
 - Аргументы  : windows ""

    Примените изменения, и теперь по нажатию выбранного вами сочетания клавиш будет появляться KRunner, выполняющий поиск только по заголовкам окон. К сожалению, такой способ не позволяет отобразить полный список всех открытых окон.
