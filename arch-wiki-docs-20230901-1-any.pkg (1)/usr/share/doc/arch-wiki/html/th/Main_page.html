    ยินดีต้อนรับสู่ ArchWiki: แหล่งข้อมูลเกี่ยวกับ Arch Linux บนเว็บสำหรับคุณ

    ไปที่ สารบัญ เพื่ออ่านข้อมูลตามหมวดหมู่

    เกี่ยวกับดิสทริบิวชั่น

    Arch Linux
    ภาพโดยรวมเกี่ยวกับ Arch Linux และสิ่งที่คุณจะพบเมื่อใช้ระบบ Arch
    คำถามที่พบบ่อย
    คำถามที่พบบ่อยและข้อมูลเกี่ยวกับ Arch
    เปรียบเทียบ Arch กับดิสทริบิวชั่นอื่น ๆ
    อธิบายความเหมือนและความต่างของ Arch เมื่อเทียบกับดิสทริบิวชั่นอื่น
    Arch ในสื่อ
    คนอื่นพูดถึงเราอย่างไร
    คู่มือการติดตั้ง
    คู่มือการติดตั้งฉบับกระชับสำหรับผู้มีประสบการณ์การใช้ Arch อยู่แล้ว
    คำแนะนำโดยทั่วไป
    สำหรับการใช้งานหลังติดตั้งและบทความยอดนิยมอื่น ๆ
    รายชื่อแอพพลิเคชั่น
    สำหรับแพคเกจอย่างเป็นทางการและจาก AUR ที่แบ่งตามหมวดหมู่

    ชุมชนของเรา

    มีส่วนร่วม
    อธิบายวิธีต่าง ๆ ที่ผู้ใช้จะมีส่วนร่วมกับชุมชนของ Arch ได้
    ArchWiki:Contributing
    คุณสามารถดูไอเดียต่าง ๆ ได้ที่นี่ ถ้าคุณพร้อมที่จะช่วยปรับปรุงวิกิ
    Help:Editing
    บทเรียนสั้น ๆ เกี่ยวกับการแก้ไขและปรับปรุงวิกิของ Arch Linux
    ชุมชนนานาชาติ
    ลิงก์ของชุมชน Arch Linux ทั่วโลก
    มารยาทในการใช้ฟอรัม
    สำหรับสมาชิกที่ใช้ฟอรัมของ Arch
    ช่อง IRC
    มาพูดคุยกับผู้ใช้ Arch คนอื่น ๆ

    ความช่วยเหลือ

    หน้าต่าง ๆ ใน ArchWiki จะถูกเขียนตามข้อกำหนดของเรา ถ้าคุณอ่านแล้วไม่เข้าใจบางส่วน ลองดูคำอธิบายที่ Help:Reading เผื่อจะเข้าใจได้ดีขึ้น

    ถ้าคุณต้องการค้นหาข้อมูล ลองใช้กล่องค้นหาที่ด้านซ้ายของหน้า หรือใช้เครื่องมือค้นหาในเว็บและกรองผลการค้นหาสำหรับวิกินี้ก็ได้ (เช่น เพิ่มตัวกรอง site:wiki.archlinux.org ต่อท้ายจากคำที่ต้องการค้นหา) สำหรับเกร็ดการค้นหาข้อมูล ลองดูที่ Help:Searching

    การลงทะเบียนจากระบบที่ไม่ใช่ Arch

    เครื่องมือ captcha ที่ใช้เพื่อลงทะเบียนบัญชี wiki ใหม่บังคับให้คุณต้องมีระบบ Arch ที่อัพเดตแล้ว อย่างไรก็ตาม เรายินดีต้อนรับคนที่ใช้ระบบปฏิบัติการอื่นด้วยเช่นกัน โดยมีข้อแม้ว่าคุณมีส่วนร่วมในเนื้อหาที่เกี่ยวกับลีนุกซ์ โดยทั่วไปเท่านั้น: สำหรับคำตอบของ captcha เราแนะนำให้คุณบู๊ตเครื่องจาก live image ล่าสุดของ Arch ใน virtual machine หรือไม่ก็ส่งอีเมล์ถึง ผู้ดูแลระบบ เพื่อขอความช่วยเหลือในด้านนี้แทน
