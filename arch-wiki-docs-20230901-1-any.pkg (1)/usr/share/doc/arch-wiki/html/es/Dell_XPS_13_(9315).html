    Edición Dell XPS 2022.

    Instalación

    En algunas unidades, la unidad NVMe está configurada para funcionar en modo RAID de forma predeterminada. Esto puede hacer que el cargador de arranque no encuentre tu disco de arranque. Para solucionar esto, presiona F2 para ingresar a la UEFI de Dell. Ve a Almacenamiento, Operación NVMe y asegúrate de que esté seleccionado el modo "Seleccionar AHCI/NVMe". Todos los demás sistemas operativos instalados antes de cambiar esta configuración deben reinstalarse después de cambiarla.

    Para arrancar correctamente el medio de instalación, deberás desactivar Arranque Seguro[enlace roto: sección no válida].

    Audio

    Esta laptop requiere el firmware sof-firmware para que la tarjeta de sonido funcione.

    Sensor de huella dactilar

    El sensor de huella dactilar se puede utilizar instalando el controlador propietario de Ubuntu lanzado por Dell y Goodix. Esto requiere una versión diferente de libfprint, libfprint-tod-gitAUR. Esta es una versión más nueva destinada a ser utilizada solo con sensores de huellas como el de el XPS.

    Advertencia: Este controlador es propietario, de código cerrado y solo se distribuye en forma binaria.

    El controlador propietario está disponible como libfprint-2-tod1-xps9300-binAUR. Alternativamente, también se puede instalar manualmente desde el repositorio de Dell extrayendo el archivo Debian y copiando su contenido.

    Advertencia: Desde que se actualizó fprintd a la versión 1.92.0-1, libfprint-tod-git y libfprint-2-tod1-xps9300-bin ya no funcionan como se espera. Esto puede evitar que GDM muestre usuarios durante el inicio de sesión. Para solucionar parcialmente este problema, debes volver a la versión 1.90.6-1 de fprintd.

    El resto del proceso es idéntico al descrito en fprint: solo asegúrate de no instalar la versión de libfprint en los repositorios oficiales, ya que entra en conflicto con libfprint-tod-git.

    Sensor de luz ambiental

    Instala iio-sensor-proxy para habilitar el brillo automático en Gnome.
