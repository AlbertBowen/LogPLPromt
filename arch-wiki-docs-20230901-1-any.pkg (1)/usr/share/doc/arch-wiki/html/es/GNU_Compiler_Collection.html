    La colección de compiladores GNU (GCC) es parte de toolchain de GNU e incluye front ends para C y C++.

    Instalación

    Instale el paquete gcc.

    Otros front-ends disponibles son:

    • gcc-ada para Ada
    • gcc-fortran para Fortran
    • gcc-go para Go
    • gcc-objc para Objective-C

    Versiones antiguas

    Las versiones antiguas de GCC están disponibles a través de los repositorios oficiales y en AUR y pueden ser útiles como curiosidad histórica, proyectos antiguos que no se pueden compilar en las versiones actuales o para probar la compatibilidad de proyectos:

    • GCC 4.3: gcc43AUR
    • GCC 4.4: gcc44AUR
    • GCC 4.5: gcc45AUR
    • GCC 4.6: gcc46AUR
    • GCC 4.7: gcc47AUR
    • GCC 4.8: gcc48AUR
    • GCC 4.9: gcc49AUR
    • GCC 5: gcc5AUR
    • GCC 6: gcc6AUR
    • GCC 7: gcc7AUR
    • GCC 8: gcc8AUR
    • GCC 9: gcc9AUR

    Se pueden encontrar otras interfaces para versiones antiguas de GCC en los repositorios oficiales y en AUR buscando gcc<version_sin_punto>, p.ej. buscando gcc9 para las interfaces de GCC 9.
