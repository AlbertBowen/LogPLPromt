    Audacious es un avanzado reproductor de audio gratuito. Se centra en la calidad del audio y soporta una amplia variedad de códecs de audio, además de ser fácilmente ampliable mediante plugins de terceros.

    Instalación

    Instale el paquete audacious. Para permitir el soporte de CDs de audio, instale libcdio.

    Nota: Algunos plugins no se cargarán a menos que se cumplan todas las dependencias opcionales del paquete audacious-plugins

    Configuración

    Plugins

    AMIDI-Plug (Reproductor MIDI)

    Para reproducir archivos MIDI y RMI, es necesario instalar el paquete fluidsynth, y también instalar los archivos de fuentes de audio, tanto freepats-general-midi como soundfont-fluid. No se requiere ninguna otra configuración en el paquete fluidsynth, pero para conocer funciones adicionales, véase FluidSynth.

    Posteriormente, el plugin se habilitará, y en el panel de configuración del plugin (Archivo, Configuración ..., panel de Plugins, pestaña Entrada, seleccione AMIDI-Plug (MIDI Player)) añada los archivos de fuentes de sonido instalados de uno en uno (Extensión .sf2) al cuadro de diálogo SoundFont, que se encuentran en /usr/share/soundfonts/.

    Consejos y trucos

    Audtool

    Audacious incluye una potente herramienta de gestión llamada Audtool que puede utilizarse para conseguir información o para controlar el reproductor.

    Por ejemplo, para conocer el título o el artista de la canción actual, ejecute las siguientes órdenes: 

    $ audtool current-song
$ audtool current-song-tuple-data artist

    También existen funciones para controlar la reproducción, manipular la lista de reproducción, el ecualizador y la ventana principal. Ejecute audtool --help para conocer la lista completa de opciones.

    Winamp

    Para añadir los temas (skins) clásicos de Winamp a Audacious, basta con copiarlos en ~/.local/share/audacious/Skins/ (sólo para el usuario) o /usr/share/audacious/Skins/ (para todo el sistema), y luego seleccionarlos en la pestaña Interfaz con temas en Preferencias. También puede arrastrar el archivo directamente a la lista de temas disponibles.
