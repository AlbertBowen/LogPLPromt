    Esta traducción de ArchWiki:Archive fue revisada el 2022-10-01. Si existen cambios puede actualizarla o avisar al equipo de traducción.

    Esta página existe como un destino de redirección para páginas archivadas.

    Si llegó aquí después de seguir un enlace a lo que se suponía que era un artículo, significa que se ha archivado: puede navegar por sus revisiones siguiendo el enlace «Redirigido desde» justo debajo del título de esta página, y luego abriendo el historial de esa página redirigida.

    Visite Category:Archive o Special:WhatLinksHere/ArchWiki:Archive para los listados de páginas archivadas.

    Si entre las páginas archivadas encuentra algunas que podrían tener sentido redirigirlas a un artículo existente, hágalo.

    Cómo archivar una página

    Antes de archivar una página, considere redirigirla a un artículo existente más adecuado (por ejemplo, redirigir traducciones antiguas o malas a la página en inglés, véase ArchWiki:Translation Team#Dealing with old translations). Si no se puede encontrar ninguno, siga este procedimiento:

    1. La página debería haber sido marcada durante al menos una semana.
    2. Elimine todos los enlaces (backlinks) que apunten a la página que se va a archivar o una de sus redirecciones, mirando su página Lo que enlaza aquí.
    3. Si la página está en los espacios de nombres "Template" (plantilla) o "Category" (categoría), debe deshabilitarse al cambiarle el nombre, cambiando los primeros dos puntos (":") por un punto y coma (";"). La página de conversación asociada, si existe, debe renombrarse también. Por ejemplo, "Template:Ejemplo" cambiaría a "Template;Ejemplo", y "Template talk:Ejemplo" cambiaría a "Talk:Template;Ejemplo".
    4. Reemplace todo el contenido con una redirección a esta misma página, y categorice la página bajo Category:Archive: 
      #redirect [[ArchWiki:Archive]]
[[Category:Archive]]
      Sugerencia: Esto se puede hacer más rápidamente con Template:AR: 
      {{subst:AR}}
    5. Corrija cualquier redirección doble que pueda resultar del archivado del artículo, es decir, todos los títulos que anteriormente estaban redireccionados a él deben archivarse igualmente como se especifica en el paso 4.
    6. Si la página archivada tiene una página de discusión asociada, debe archivarse con este mismo procedimiento, comenzando desde el paso 2.
    7. Si la página archivada tiene traducciones asociadas, se deben archivar con este mismo procedimiento, comenzando desde el paso 2.
