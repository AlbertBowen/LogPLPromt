    • pacman

    pkgfile es una herramienta para buscar archivos de paquetes en los repositorios oficiales.

    Sugerencia: pacman tiene una funcionalidad similar incorporada.

    Instalación

    Instale el paquete pkgfile. De manera alternativa, puede instalar la versión de desarrollo con el paquete pkgfile-gitAUR.

    La base de datos de pkgfile puede entonces ser sincronizada con la orden: 

    # pkgfile -u

    Utilización

    Para buscar un paquete que posea el archivo makepkg: 

    $ pkgfile makepkg
    core/pacman

    Para listar todos los archivos proporcionados por archlinux-keyring: 

    $ pkgfile -l archlinux-keyring
    core/archlinux-keyring usr/
core/archlinux-keyring usr/share/
core/archlinux-keyring usr/share/pacman/
core/archlinux-keyring usr/share/pacman/keyrings/
core/archlinux-keyring usr/share/pacman/keyrings/archlinux-revoked
core/archlinux-keyring usr/share/pacman/keyrings/archlinux-trusted
core/archlinux-keyring usr/share/pacman/keyrings/archlinux.gpg

    Este último es similar a pacman -Ql (véase Pacman (Español)#Consultar la base de datos de los paquetes), excepto que se aplica a los paquetes remotos.

    Orden no encontrada

    Véase Bash (Español)#No se encontró la orden, Zsh#pkgfile "command not found" handler y Fish#The "command not found" hook.

    Actualizaciones automáticas

    pkgfile incluye un servicio systemd y un temporizador para sincronizar automáticamente la base de datos de pkgfile. Para permitir las actualizaciones automáticas active pkgfile-update.timer.

    Por defecto, pkgfile se actualizará a diario. Para cambiar el horario, edite el archivo de unidades.
